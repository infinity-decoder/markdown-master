Modular, enterprise-grade LMS, Quiz, Code, and Markdown system for WordPress.

== Description ==

Cotex is a modular plugin architecture that lazy-loads features only when needed.

**Modules:**
*   **LMS Engine**: Full course management.
*   **Quiz Engine**: Advanced quizzes with various question types.
*   **Code Blocks**: Syntax highlighting with PrismJS.
*   **Markdown Studio**: Markdown authoring and rendering.

== Installation ==

1.  Upload the plugin files to the `/wp-content/plugins/cortex` directory, or install the plugin through the WordPress plugins screen directly.
2.  Activate the plugin through the 'Plugins' screen in WordPress.
3.  Navigate to the Cotex dashboard to enable modules.

== Frequently Asked Questions ==

= How do I enable a module? =

Go to the Cotex dashboard page and toggle the switch for the desired module.
